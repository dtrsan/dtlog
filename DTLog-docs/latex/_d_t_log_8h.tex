Redirects stderr output to a file. If append is set to true, redirect is done with mode \char`\"{}a\char`\"{} which will opens file for output at the end of a file. The file is created if it does not exist. If append is set to false, redirect is done with mode \char`\"{}w\char`\"{} which will create an empty file for output operations. If a file with the same name already exists, its contents are discarded and the file is treated as a new empty file. 
\begin{DoxyParams}{Parameters}
{\em file} & The file path. \\
\hline
{\em append} & If true output will be appended to the existing file, otherwise new file will be created and existing will be overwritten. \\
\hline
\end{DoxyParams}
